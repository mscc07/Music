Welcome to LITESOPH
=====================
Starting LITESOPH using the command ( see :ref:`usage` ) for the first time, 
you will get into the following Welcome panel:

.. image:: ./LS_home.png
   :width: 800
   :alt: LITESOPH_home


* Create LITESOPH Project : Create a new LITESOPH project.
* :ref:`Open LITESOPH Project` : Open the :ref:`predefined` for  the further Calculations.
* About LITESOPH : Click on this button to see "About LITESOPH".
